# Admonitions 提示框功能演示

本文演示 QuillStack 博客系统支持的 Admonitions 提示框语法。

## 什么是 Admonitions？

Admonitions（提示框）是一种用于在文档中突出显示重要信息的 Markdown 扩展语法。它可以帮助读者快速识别不同类型的提示信息。

## 支持的类型

QuillStack 支持以下 5 种提示框类型：

### INFO - 信息提示

> [!INFO]
> 这是一个**信息提示框**，用于展示一般性的提示信息。支持 **Markdown** 语法，如*斜体*、`代码`和[链接](https://example.com)。

### WARNING - 警告提示

> [!WARNING]
> 这是一个**警告提示框**，用于提醒用户注意可能的问题。支持嵌套列表：
> - 第一项
> - 第二项
> - 第三项

### TIP - 技巧提示

> [!TIP]
> 这是一个**技巧提示框**，用于分享实用的技巧和建议。可以包含代码示例：
> ```bash
> npm install
> npm run dev
> ```

### DANGER - 危险提示

> [!DANGER]
> 这是一个**危险提示框**，用于警告用户可能造成严重后果的操作。请谨慎操作！

### NOTE - 备注提示

> [!NOTE]
> 这是一个**备注提示框**，用于补充说明相关信息。可以包含引用：
> > 这是嵌套的引用内容。

## 使用语法

```markdown
> [!INFO]
> 这是一个信息提示框

> [!WARNING]
> 这是一个警告提示框

> [!TIP]
> 这是一个技巧提示框

> [!DANGER]
> 这是一个危险提示框

> [!NOTE]
> 这是一个备注提示框
```

## 多行内容支持

提示框支持多行内容，每行都需要以 `>` 开头：

> [!TIP]
> 第一行内容
> 第二行内容
> 第三行内容

## 嵌套 Markdown

提示框内部完全支持 Markdown 语法：

> [!INFO]
> **加粗文本**、*斜体文本*、`行内代码`、[外部链接](https://example.com)
>
> - 无序列表项 1
> - 无序列表项 2
>   - 嵌套列表项
>   - 嵌套列表项
>
> 1. 有序列表项 1
> 2. 有序列表项 2

## 总结

Admonitions 是增强 Markdown 文档可读性的强大工具，合理使用可以让你的文档更加专业和易读。
